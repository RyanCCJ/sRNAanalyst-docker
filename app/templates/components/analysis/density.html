Density Distribution

Density Distribution helps observe the distribution of read count density across different datasets, interest lists, and regions, presented in a box plot. You need to input a specific format of reference CSV file to specify the analysis region coordinates (1-based), with the column name representing the designated region name.

For example, to observe the read density across the entire length of each mRNA, you can input an interval named "ALL," including each length (1-N). If you want to compare the distribution of 5'UTR, CDS, and 3'UTR, you need to input a file containing information about the coordinates of these three regions.

Please input a valid data for density analysis.
