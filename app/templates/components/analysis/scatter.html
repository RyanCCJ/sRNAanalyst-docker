Scatter Plot

The scatter plot analysis focuses on observing data from one or multiple pairs of wild-type and mutant samples. It allows further comparison of the changes in specific gene lists under different experimental conditions based on the selected observation set. To use this feature, please select Paired Data on the input Dataset page and upload the wild-type and mutant data for comparison.

The reference CSV file needs to include the coordinates of the regions to be observed for each target RNA; refer to the Density page for further instructions.

Please input a valid pair of data (control and mutant) for scatter analysis.
