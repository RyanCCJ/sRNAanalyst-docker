Instruction

Upload the data table of the experiment for downstream analysis. The data table must be the following format:

  • Organized in read-reference format using alignment software, including fields such as read_id, ref_id, read_count, init_pos, end_pos, etc. Not necessarily in order.
  • The file should be in CSV format (Comma-Separated Values), with comments indicated by the "#" symbol at the beginning and including headers for each field.

You can use the Preprocessing Workflow or Helper Tool on this website to generate data tables. The Workflow accepts a FASTQ file or its compressed version, while the Helper Tool accepts a SAM/BAM file.

Settings

Select and upload either single data or paired data (control and mutant) for further analysis. Please note that fold-change and scatter plot analyses can only be performed using paired data.
