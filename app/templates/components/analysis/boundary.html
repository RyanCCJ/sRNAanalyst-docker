Head & Tail Distribution

The Head and Tail Distribution helps to observe the read count distribution of data at the head and tail regions. In addition to the default 100 nt length, users can also set the desired length or specific positions for observation. It mainly includes three types of analysis: binding ratio, read counts, and read counts distribution.

The reference CSV file should include the coordinate information for the start and end of each target RNA, with columns named 'head' and 'tail'.
